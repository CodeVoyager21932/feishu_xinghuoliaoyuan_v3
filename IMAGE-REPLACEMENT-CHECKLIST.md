## 🎯 设计规范速查

### 图标规范
```
尺寸: 256x256px (@2x) 或 512x512px (@3x)
格式: PNG-24 (透明背景)
大小: < 50KB
风格: 统一（线性 或 面性）
颜色: 红色主题 (#D32F2F)
```

### 头像规范
```
尺寸: 400x500px (竖版) 或 500x500px (方形)
格式: JPG (照片) 或 PNG (插画)
大小: < 200KB
风格: 统一（真实照片 或 插画）
```

### 第1步: 设计/获取图片

## 📞 遇到问题？

### 常见问题快速解决

**Q: 图片不显示？**
- 检查文件名大小写
- 检查文件路径
- 重新编译小程序

**Q: 图片模糊？**
- 使用更高分辨率
- 至少 2x (256x256px)

**Q: 文件太大？**
- 使用 TinyPNG 压缩
- 降低 JPG 质量到 80%

**Q: 风格不统一？**
- 使用同一个图标库
- 保持相同的线条粗细
- 统一颜色主题
