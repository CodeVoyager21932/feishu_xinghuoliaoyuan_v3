# 日签背景图片目录

## 说明
此目录用于存放日签功能的背景图片。

## 图片规格
- **尺寸**: 750px × 1334px (2:3.56 竖版海报比例)
- **格式**: JPG/PNG
- **风格**: 国潮风、红色主题、历史感

## 建议图片内容
1. 红色革命遗址（如延安宝塔山、井冈山、遵义会议旧址）
2. 历史文物（如红船、长征路线图）
3. 抽象的红色水墨背景
4. 党旗、五角星等元素的艺术化处理

## 命名规范
- `bg-001.jpg` ~ `bg-030.jpg`
- 或使用主题命名：`bg-changzheng.jpg`（长征主题）

## 默认图片
如果没有准备图片，系统会使用纯色渐变背景作为占位。
